# Gerenciador de Tarefas Ágil

## Descrição do Projeto
Este projeto consiste no desenvolvimento de um sistema simples de gerenciamento de tarefas, criado como atividade prática da disciplina de Engenharia de Software. O sistema tem como objetivo organizar tarefas de forma clara, utilizando conceitos de metodologias ágeis e ferramentas de versionamento e controle de qualidade.

## Objetivo
O objetivo do projeto é permitir o cadastro, visualização, atualização e exclusão de tarefas, facilitando o acompanhamento do fluxo de trabalho de uma equipe de forma simples e organizada.

## Escopo Inicial
No escopo inicial, o sistema permite:
- Criar tarefas
- Listar tarefas cadastradas
- Atualizar informações das tarefas
- Excluir tarefas
Cada tarefa possui título, descrição e status.

## Metodologia Utilizada
Foi utilizada a metodologia ágil Kanban, com apoio do GitHub Projects, organizando as tarefas nas colunas "To Do", "In Progress" e "Done". Essa abordagem permite visualizar o andamento do projeto e facilita o controle das atividades ao longo do desenvolvimento.

## Como Executar o Projeto
1. Clone o repositório:

    git clone https://github.com/GabMasca/gerenciador-tarefas

2. Acesse a pasta do projeto:

    cd gerenciador-tarefas

3. Instale as dependências:

    pip install -r requirements.txt

4. Execute os testes automatizados:

    pytest

#  Mudança de Escopo
Durante o desenvolvimento do projeto, foi identificada a necessidade de melhorar a organização das tarefas. Por isso, foi adicionada a funcionalidade de prioridade das tarefas, permitindo classificá-las como baixa, média ou alta. Essa mudança foi registrada no Kanban e implementada respeitando os princípios da metodologia ágil.
